## Sobre

### Projeto de um jogo da memória feito com typescript no React temático do jogo Lost Records: Bloom

## Como usar

### 1. Clonar o projeto em sua máquina:

```
git clone https://github.com/isamsm/gameflix
```

### 2. Instalar dependências:

```
npm i
```

### 3. Rodar o projeto:

```
npm start
```
